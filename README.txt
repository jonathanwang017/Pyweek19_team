Walls - The Maze Room
=====================

Entry in PyWeek #19  <https://www.pyweek.org/19/>
URL: https://pyweek.org/e/first_attempt
Team: bandcamp
Members: jonw665, myonkorux, mchawla33


Dependencies
------------

This game was developed with Python 2.7.3 and Pygame 1.9.1. These can be downloaded at:

  https://www.python.org/download/releases/2.7.6/
  http://www.pygame.org/download.shtml

On Ubuntu:

  sudo apt-get install python python-pygame


Running the Game
----------------

On Windows run walls_pyweek19.exe in the windows_dist folder

On Mac run walls_pyweek19.app in the mac_dist folder

Otherwise open a terminal / console and "cd" to the source folder in the game directory and run:

  python walls_pyweek19.py 




How to Play the Game
--------------------

left/right/up/down to move

Get out of the room
